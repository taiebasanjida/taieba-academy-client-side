# Taieba Academy (Client)

Live URL: https://your-live-client-url.example.com

## Features

- Modern single-page app with React Router and protected routes
- Firebase Authentication (Email/Password + Google) with preserved session on reload
- Courses: browse, filter by category, view details, enroll with toast feedback
- Instructor Dashboard: add, list, update, and delete own courses
- Responsive UI with Tailwind CSS, theme toggle (dark/light), and animations (Framer Motion, AOS)

## Getting Started

1. Install dependencies: `npm install`
2. Create `.env` with Firebase and API base URL (see `.env.example`)
3. Run dev server: `npm run dev`

## Tech

React, Vite, Tailwind, Firebase Auth, React Query, Axios, Framer Motion, AOS.
